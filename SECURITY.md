# Security Policy

| Version | Supported          |

## Limititations

**Secure AI Transcriber** is a **Client-Side Application**. 

1.  **API Key Visibility**: The API Key is present in the source code. This is **by design**.
2.  **XSS & CSP**: We have implemented strict Content Security Policy (CSP).

## Reporting a Vulnerability

If you discover a new security vulnerability, please email: security@example.com
